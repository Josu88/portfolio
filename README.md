# Portfolio Personal

Este proyecto es un portfolio personal, en el muestro información personal y  de contacto sobre mi(email,linkedin), los proyectos que hice o que participe en ellos y mi curriculum en canva. 

## Instalación

Para poder ver el porfolio:

1.Descargaremos el proyecto en rar o haremos un git clone de la url que pone.</br>
2.Ejecutaremos el comando npm install para instalar las dependencias necesarias para su funcionamiento.</br>
3.Ejecutaremos el comando npm start para lanzar la aplicación y poder visualizarla.</br>
